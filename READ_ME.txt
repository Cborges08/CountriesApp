STEP A: Set up Backend
1) Open a terminal and go into "backend" folder:
   cd backend
2) Install dependencies:
   npm install
3) Ensure .env file has correct variables (see above).
4) Start dev server:
   npm run dev
   (Runs on http://localhost:4000 by default.)

STEP B: Set up Frontend
1) In a NEW terminal, go into "frontend" folder:
   cd frontend
2) Install dependencies:
   npm install
3) Ensure .env.local file has correct variables (see above).
4) Start dev server:
   npm run dev
   (Runs on http://localhost:3000 by default.)

STEP C: Access the App
Open your browser to http://localhost:3000 
You should see the Home Page with a list of countries.
